# 🏆 Candy Coded – Gamified Python Learning Experience  
🏆 First Place – JPMorgan Chase Code for Social Good Award (Technica)

## 📌 Overview
Candy Coded is a gamified mobile learning experience designed to introduce beginner programming concepts through interactive storytelling and problem-solving.

The application follows Princess Cherri on a journey from Twisted Tower back to Cotton Candy Castle, where users progress through levels by solving Python-based logic challenges.

The goal is to make programming more approachable, engaging, and accessible for beginners, especially young learners.

---

## 🎯 Problem
Traditional programming education can feel intimidating and abstract for beginners, leading to low engagement and difficulty retaining foundational concepts.

---

## 💡 Solution
We designed a narrative-driven learning experience that combines storytelling with structured Python logic challenges. Users learn by progressing through an interactive game environment rather than traditional instruction.

---

## 🧠 Key Features
- Narrative-based progression system
- Interactive Python logic challenges
- Gamified learning structure
- Beginner-friendly programming concepts
- Mobile-first experience design

---

## 🏆 Outcome
- First Place Winner – JPMorgan Chase Code for Social Good Award
- Recognized for making STEM education more accessible and engaging

# 💰 Financial Literacy & Budgeting App

## 📌 Overview
A mobile application designed to help users improve financial literacy by tracking spending, managing budgets, and visualizing financial habits in an intuitive and user-friendly interface.

---

## 🎯 Problem
Many individuals struggle with budgeting and lack clear visibility into their spending patterns, making it difficult to develop strong financial habits.

---

## 💡 Solution
We built a simple and intuitive budgeting experience that helps users track expenses, categorize spending, and gain better awareness of their financial behavior over time.

---

## 🧠 Key Features
- Expense tracking and categorization
- Budget creation and monitoring
- Financial summaries and insights
- Clean, user-friendly mobile design
